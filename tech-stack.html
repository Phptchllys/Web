SShopee Forms
หน้าแรกเข้าสู่ระบบสมัครใหม่ฟอร์มสินค้าฟอร์มคำสั่งซื้อTech Stack

แผนพัฒนาระบบ

ตามที่กำหนด: React + Firebase Hosting + Firebase Authentication + Firestore/Realtime Database + Cloud Storage

1) React

พัฒนา UI เป็น Components เช่น LoginPage, RegisterPage, ProductForm, OrderForm เพื่อบริหาร state และ routing ได้ง่าย

2) Firebase Hosting

นำแอป React ที่ build แล้วขึ้นระบบจริง (Production) พร้อม HTTPS และรองรับ deployment workflow

3) Firebase Authentication

รองรับ สมัครสมาชิก / เข้าระบบ / ลืมรหัสผ่าน และสามารถขยาย social login ได้

4) Firestore / Realtime Database

ใช้ฐานข้อมูล NoSQL เก็บข้อมูลสินค้า คำสั่งซื้อ และข้อมูลผู้ใช้แบบเรียลไทม์

5) Firebase Cloud Storage

ใช้อัปโหลดและจัดเก็บรูปภาพ เอกสาร และไฟล์ต่าง ๆ ที่แนบกับสินค้า/คำสั่งซื้อ

เชื่อมต่อพร้อมใช้งานในหน้า Login/Register

หน้า login.html และ register.html เชื่อมกับ Firebase Authentication ผ่านไฟล์ firebase-auth.js แล้ว เหลือเพียงใส่ค่า Firebase Config จริงก่อนใช้งาน
